## Version Management
This project uses **jujutsu** (`jj`) for version control.

Common jj commands:
- `jj status` — working copy status
- `jj diff` — pending changes
- `jj describe -m "message"` — set commit description
- `jj new` — start a new change
- `jj log` — history
