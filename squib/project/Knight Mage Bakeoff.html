02 Sword stance · passive states

When all is well.

Sword drawn, weight on back foot, hood low. Cloak hangs heavy. Eye-glow steady. The stoic mode.

03 Pyromancy stance · active states

When something burns.

Sword sheathed. Free palm out, ember orb hovering. Forward-lean, cloak lifted by heat shimmer. Eye-glow flares — drifting embers rise from the body. The incantatory mode.

04 Bonfire · resting state

When the session ends.

Kneeling at a bonfire, sword stabbed in earth beside. Plate dimmed. Soft Z's drift. The agent is offline — squib is resting at the bonfire too.

The weapon-swap rule

  • idle / working — Longsword. grounded. cloak heavy.
  • thinking / attention — Pyromancy. lean forward. embers rise.
  • error — Pyromancy fizzles. spell sputters out.
  • sleeping / session-end — Bonfire kneel. weapon stabbed in earth.

Same character. Same silhouette. The weapon tells the user what mood the agent is in without a single word of UI copy.

Animation hooks (per pose)

  • cloak sway — base ambient on every state
  • chest breathe — vertical bob, ~3s loop
  • eye-glow pulse — ember red, faster on active
  • weapon-specific FX — slash arc, ember orb, drifting particles, bonfire flicker
  • helmet tilt — head turns toward cursor on hover (handoff trick)
